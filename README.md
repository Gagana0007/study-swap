## 🌍 Overview
**Study Swap** is a modern, responsive web application that connects learners worldwide for **peer-to-peer skill exchange**. Instead of paying for courses, users trade their existing tech skills for new ones. Teach JavaScript, learn React. Share Python, get Figma help. Completely free, collaborative, and secure.

> 💡 *Built for students, developers, and self-taught creators who believe in community-driven learning.*

---

## ✨ Features
- 🔐 **Secure Authentication** (Email/Password via Supabase Auth + JWT)
- 🧠 **Smart Skill Matching** (PostgreSQL array overlap algorithm)
- 📹 **Live Video & Chat Sessions** (WebRTC via PeerJS + Supabase Realtime)
- 📅 **Session Scheduling & Management**
- 📊 **Progress Tracking & Analytics**
- ⭐ **Review & Rating System**
- 📱 **Fully Responsive** (Mobile, Tablet, Desktop)
- 🌐 **Custom Domain & HTTPS** Ready

## 🚀 How It Works
1. **Sign Up** → Create a profile with skills you `Teach` & `Want to Learn`
2. **Get Matched** → Algorithm finds users with complementary skills
3. **Schedule** → Pick a time, choose video/chat mode
4. **Swap Skills** → Live session with screen sharing, chat, and collaborative notes
5. **Rate & Review** → Build trust in the community
